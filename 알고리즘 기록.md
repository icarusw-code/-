## 알고리즘 기록



level1. 행렬의 덧셈

zip 내장 함수 사용 => 병렬로 출력 가능

```
>>> for number, upper, lower in zip("12345", "ABCDE", "abcde"):
...     print(number, upper, lower)
...
1 A a
2 B b
3 C c
4 D d
5 E e
```

```
def solution(arr1, arr2):
    answer = [[c + d for c, d in zip(a, b)] for a, b in zip(arr1,arr2)]
    return answer
```
